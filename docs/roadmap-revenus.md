# Roadmap Trafic & Revenus — Projection Honnête

> Contexte : Site neuf, 0 autorité, département Eure-et-Loir (~430k hab.)
> 330 pages money + 10 trade hubs + 33 city hubs + blog
> Monétisation : affiliation leads (Travaux.com, Effy)

---

## Hypothèses de calcul

- **Taux de conversion visiteur → lead** : 2-4% (standard pour un comparateur de prix)
- **Taux d'acceptation plateforme** : 50-70% (toutes les plateformes rejettent une partie des leads : doublons, numéro invalide, hors zone, etc.)
- **CPA moyen Travaux.com** : 8-12€/lead accepté
- **CPA moyen Effy** (leads énergie) : 15-25€/lead accepté
- **CPA moyen pondéré** : ~12€/lead (mix tous métiers)

---

## Mois 1-3 : L'indexation (quasi 0€)

| | Réaliste |
|---|---|
| Visites/mois | 0 → 100 |
| Leads/mois | 0 → 2 |
| Revenus/mois | 0 → 20€ |

**Ce qui se passe :** Google crawle et indexe les 394 pages. Tes premières impressions apparaissent dans Search Console, surtout sur des requêtes ultra long-tail ("prix toiture brunelles" que personne ne cherche). Pas de positions page 1 sur les requêtes qui comptent. C'est normal et c'est la phase la plus frustrante.

**Ce que tu dois faire :**
- Soumettre le sitemap dans Search Console
- Vérifier que toutes les pages sont indexées (demander l'indexation manuellement pour les pages prioritaires)
- Écrire 2-3 articles de blog supplémentaires pour montrer de l'activité à Google
- T'inscrire sur Travaux.com et Effy

## Ce qui peut mal tourner (scénario pessimiste)

- **Google tarde à indexer** : sur un domaine neuf sans backlinks, l'indexation de 394 pages peut prendre 3-6 mois
- **Taux de conversion < 2%** : si le formulaire n'inspire pas confiance ou si le trafic est trop informationnel
- **Plateformes rejettent trop de leads** : leads hors zone, numéros invalides, doublons
- **Un concurrent local apparaît** : peu probable mais possible
- **Google update** : une mise à jour de l'algorithme peut bousculer les positions

## Ce qui peut accélérer les choses (scénario optimiste)

- **Backlinks locaux** : si tu obtiens des liens de la CCI, de journaux locaux, d'annuaires pro
- **Google Business Profile** : même pour un site d'info, ça aide le signal local
- **Articles invités** sur des blogs locaux/régionaux
- **Réseaux sociaux locaux** : groupes Facebook Eure-et-Loir
- **Ajout de départements voisins** : dupliquer le modèle sur le 41 (Loir-et-Cher), 45 (Loiret), 91 (Essonne) — ça multiplie le trafic potentiel

## Comment dépasser le plafond

L'Eure-et-Loir seul a un plafond naturel (petit département, ~430k hab.). Pour scaler :

1. **Départements voisins** : Dupliquer le modèle sur 3-4 départements = x4 le potentiel
2. **Vente directe aux artisans** : Au lieu de passer par Travaux.com à 12€/lead, vendre directement aux artisans locaux à 20-30€/lead
3. **Abonnements artisans** : Forfait mensuel 100-200€/mois pour recevoir X leads/mois → revenu récurrent plus stable
4. **Contenu national** : Articles "prix [trade] m2" qui ciblent le trafic national (volume x100 mais concurrence x50)

---

## Conclusion honnête

Ce n'est pas un projet qui va te rendre riche. C'est un projet de revenu complémentaire passif qui, s'il est bien exécuté, peut générer **700-1200€/mois en année 2** sur un seul département, avec très peu de maintenance une fois en place.

Le vrai potentiel est dans la duplication : le même site adapté à 5 départements = 3500-6000€/mois. Et la transition vers la vente directe de leads aux artisans (sans intermédiaire) multiplie les marges par 2-3.
